# Deepl translater for kde widgets

## Development

Just fork or clone the project :


run the command on terminal 
`git clone https://github.com/samirgaire10/com.samirgaire10.Deepl.git`

move the file to `/home/.local/share/plasma/plasmoids/`

and run the command  
`plasmawindowed  com.samirgaire10.Deepl/`
